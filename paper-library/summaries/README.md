# Reading Highlights

Create one markdown file per paper using the paper ID from `manifest/paper-manifest.json`.

## Template

Use this structure:

```md
# Paper Title

- Venue:
- Year:
- Topic fit:
- Status:

## One-line takeaway

## What problem it claims to solve

## Reading highlights

- ...
- ...

## Section notes

### Abstract
### Introduction
### Method
### Experiments
### Limitations

## Figures to inspect

- Figure 1:
- Figure 2:

## Why it matters for temporal hallucination

## What still feels unresolved

## Next action

```

## Naming

- File name: `{paper-id}.md`
- Optional figure crops go in `../figures/{paper-id}/`

## Minimum bar

Each summary should make the paper readable even if the original PDF is skimmed later.
